Site Name: Blaq Code - My Journey As A Web Developer

This name was chosen because when I started my journey as a web developer, someone mockingly gave me the nickname Blaq Code, which name I have turned to something beautiful

Site Purpose:

The purpose of this site is for how I started learning to code; where I started from and how I am currently doing. It is going to be a static blog site that will have some articles about my journey as a web developer, and some articles on web design topics.

Scenarios:

  • How did you get started with learning to code, and what resources helped you the most as a beginner?
  • What are some simple web design tips I can use to make my portfolio or personal site look better?

Typography Plan

Fonts Used:

  • Noto Serif – used for headings
  • Inter – used for body text, navigation, and captions

Font Usage Breakdown:

Font Use Case Style
Noto Serif Headings (.heading-xl, .heading-lg, .heading-md) Elegant, classic serif for strong visual hierarchy
Inter Body text, nav, captions (.text-body, .caption) Modern, clean sans-serif for easy screen reading
